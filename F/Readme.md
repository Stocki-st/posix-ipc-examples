## how to build

clean with
```
make clean
```

format using astyle
```
make format
```

build everything
```
make all
```


## how to run:

1. start menu: `./menu`
2. start compute: `./compute`
3. start display: `./display`

now menu reads the input from the cmdline, compute transforms it to uppercase and display will print it.
